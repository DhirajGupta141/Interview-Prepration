Given a binary tree A of integers. Return an array of integers representing the right view of the Binary tree.

Right view of a Binary Tree: is a set of nodes visible when the tree is visited from Right side.



Problem Constraints
1 <= Number of nodes in binary tree <= 105

0 <= node values <= 109



Input Format
First and only argument is an pointer to the root of binary tree A.



Output Format
Return an integer array denoting the right view of the binary tree A.



Example Input
Input 1:

        1
      /   \
     2    3
    / \  / \
   4   5 6  7
  /
 8 
Input 2:

    1
   /  \
  2    3
   \
    4
     \
      5


Example Output
Output 1:

 [1, 3, 7, 8]
Output 2:

 [1, 3, 4, 5]
_____________________________________________Solution______________________________
public class Solution 
{
    public ArrayList<Integer> solve(TreeNode root)
    {
        ArrayList<Integer> ans=new ArrayList<>();
        
        if(root==null)
        {
            return ans;
        }
        
        Deque<TreeNode> mq=new ArrayDeque<>();   // ArrayDeue is used here so that I can
        Deque<TreeNode> cq=new ArrayDeque<>(); //get  and add  the elements in O(1)
        
        mq.add(root);
        
        while(!mq.isEmpty())
        {
            int size=mq.size();
            ans.add(mq.getLast().val); // Getting the last element of that level
            
            for(int i=0;i<size;i++)
            {
                TreeNode temp=mq.poll();
                
                if(temp.left!=null)
                {
                    cq.add(temp.left);
                }
                if(temp.right!=null)
                {
                    cq.add(temp.right);
                }
                
            }
            
            if(mq.isEmpty())
            {
                mq=cq;
                cq=new LinkedList<>();
            }
            
        }
        
        return ans;
        
    }
    
}
