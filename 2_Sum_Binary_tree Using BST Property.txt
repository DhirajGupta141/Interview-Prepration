Given a binary search tree T, where each node contains a positive integer, and an integer K, you have to find whether or not there exist two different nodes A and B such that A.value + B.value = K.

Return 1 to denote that two such nodes exist. Return 0, otherwise.

Notes

Your solution should run in linear time and not take memory more than O(height of T).
Assume all values in BST are distinct.
Example :

Input 1: 

T :       10
         / \
        9   20

K = 19

Return: 1

Input 2: 

T:        10
         / \
        9   20

K = 40

Return: 0

-----------------------------------------------SOLUTION-------------------
public class Solution 
{
   void helper(TreeNode root,ArrayList<Integer> anslist)
   {
       if(root==null)
       {
           return;
       }
       
       helper(root.left,anslist);
       anslist.add(root.val);
       helper(root.right,anslist);
       
   }
    public int t2Sum(TreeNode root, int key)
    {
        if(root==null)
        {
            return 0;
        }
        ArrayList<Integer> anslist=new ArrayList<Integer>();
        helper(root,anslist);
        
        int i=0;
        int j=anslist.size()-1;
        while(i<j)
        {
            if(anslist.get(i)+anslist.get(j)==key)
            {
                return 1;
                
            }
            else if(anslist.get(i)+anslist.get(j)>key)
            {
                j--;
            }
            else
            {
                i++;
            }
        }
        
        return 0;
        
    }
}
