Find the longest increasing subsequence of a given array of integers, A.

In other words, find a subsequence of array in which the subsequenceâ€™s elements are in strictly increasing order, and in which the subsequence is as long as possible.
This subsequence is not necessarily contiguous, or unique.
In this case, we only care about the length of the longest increasing subsequence.


Input Format:

The first and the only argument is an integer array A.
Output Format:

Return an integer representing the length of the longest increasing subsequence.
Constraints:

1 <= length(A) <= 2500
1 <= A[i] <= 2000
Example :

Input 1:
    A = [1, 2, 1, 5]

Output 1:
    3
    
Explanation 1:
    The sequence : [1, 2, 5]

Input 2:
    A = [0, 8, 4, 12, 2, 10, 6, 14, 1, 9, 5, 13, 3, 11, 7, 15]
    
Output 2:
    6

Explanation 2:
    The sequence : [0, 2, 6, 9, 13, 15] or [0, 4, 6, 9, 11, 15] or [0, 4, 6, 9, 13, 15]
________________________________________________________________________SOLUTION___________________________________________________________________
public class Solution
{
    // DO NOT MODIFY THE ARGUMENTS WITH "final" PREFIX. IT IS READ ONLY
    int longestInceasingSubsequnce(int []a)
    {
        int om=Integer.MIN_VALUE;  //Overall max that is answer
        int t[]=new int[a.length]; // dp array for storing lis till me
        for(int i=0;i<a.length;i++)
        {
            int cur_max=0;   //initially the max at that index will be zero
            for(int j=0;j<i;j++)
            {
                if(a[i]>a[j])  // my value is greater than checking value
                {
                    if(t[j]>cur_max)  //getting the best value at i using dp array
                    {
                        cur_max=t[j];
                    }
                }
            }
            t[i]=cur_max+1;  // adding my count as well  
            if(t[i]>om)    //getting the overall answer
            {
                om=t[i];
            }
        }
        return om;
    }
    
    public int lis(final int[] a)
    {
        if(a==null ||a.length==0)
        {
            return 0;
        }
        
        int ans=longestInceasingSubsequnce(a);
        
        return ans;
    }
}
